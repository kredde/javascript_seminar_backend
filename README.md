# Javascript Seminar Backend

## Get started

- Set up a local mongodb instance
- Create a `.env` using the `.env.example`
- Install dependencies `npm i`
- Start the application with `npm run dev`
- Visit Swagger at `http://localhost:5000/v1/docs/`
